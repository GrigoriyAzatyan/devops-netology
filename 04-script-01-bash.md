## 1. Есть скрипт:

`a=1`  
`b=2`  
`c=a+b`  
`d=$a+$b`  
`e=$(($a+$b))`  
Какие значения переменным c,d,e будут присвоены? Почему?

**Ответ:**  
- Значением c будет строка "**a+b**";   
- Значением d будет строка "**1+2**";  
- Значением e будет **число 3**.
